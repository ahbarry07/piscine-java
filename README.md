## Overview

The Java Piscine ran from **September 2 to September 23, 2024** and was successfully completed. ## What I Learned

### Java Fundamentals
- Strongly typed, compiled language model (JVM)
- Primitive types vs reference types
- Control flow, exception handling (`try/catch/finally`)
- Command-line I/O and file operations

### Object-Oriented Programming
- Four pillars: encapsulation, abstraction, inheritance, polymorphism
- Class design: constructors, getters/setters, access modifiers
- Abstract classes vs interfaces — when and why to use each
- Method overloading and overriding

### Collections Framework
- `ArrayList`, `LinkedList`, `HashMap`, `HashSet`, `TreeMap`, `Queue`
- Iterators and the `for-each` pattern
- Comparable and Comparator for custom sorting
- Generics for type-safe collections

### Java Standard Library
- `java.lang`: `String`, `Math`, `System`, `Object`
- `java.util`: Collections, Date/Time, Optional
- `java.io`: Streams, readers/writers, file operations

### Design Patterns
- **Creational**: Singleton, Factory Method, Builder
- **Structural**: Adapter, Decorator, Facade
- **Behavioral**: Observer, Strategy, Command, Iterator
